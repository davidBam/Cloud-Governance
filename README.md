## Wat is het, de 7 kern methodologieën van het CAF

![Microsoft Cloud Adoption Framework 7 kern methodologieën](https://learn.microsoft.com/en-us/azure/cloud-adoption-framework/caf-overview.png)

Het CAF is een praktisch gefocusde verzameling van richtlijnen, werkprocessen en best practises die organisaties (klein en groot, in alle sectoren) kunnen helpen met het plannen, uitrollen en beheren van hun resources in de Azure cloud. Het doel van het framework is om de bedrijfsdoelen te behalen en daarbij risoco te minimaleren en 

Het framework bestaat uit 7 kern methodologieën:

foundation: [sequentially ]
- Strategy
- Plan
- Ready
- Adopt

Operationele: [parallel ]
- Govern
- Secure 
- Manage
